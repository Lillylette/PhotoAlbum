# ТЗ по проекту приложенияпо управлению личными фотографиями "Фотоальбом"

### Основные функциональные требования

1. Открытие файлов изображений 
   
   Возможность открывать имеющиеся в галерее файлы изображений форматов .jpeg, .png, .tiff, .bmp.
   
   Возможность открывать новые скачанные фотографии с внешнего носителя
   
   Внесение новых фотографий в галерею

2. Обработка имеющихся и новых фотографий
   
   Переименование файлов по пользовательским шаблонам
   
   Добавление к изображениям комментариев в формате файлов .csv
   
   Применение к выбранным фотографиям фильтров.
   
   Добавление на изображения надписей
   
   Геометрические преобразования фотографий (обрезка, изменение размера, повороты, отражения).
   
   Добавление рамки и виньетки.

3. Просмотр фотографий в отдельном окне.
   
   ### 

### Архитектура и дизайн приложения

#### Основные модули приложения

Первое окно:

1. Виджет для добавления новых фотографий

2. Виджет для входа в базу данных (галерею)

Окно добавления фотографий:

1. Виджет открытия папки с файлами

2. Виджет со списком открываемых фотографий

3. Виджет удаления фотографий

4. Виджет добавления фотографий в галерею

Окно просмотра выделенной фотографии

Окно переменования файлов:

1. Виджет для ввода шаблона названия файлов (префикс, номер по порядку)

2. Виджет добавления фотографий в галерею   

Окно открытия фотографий в галерее:

1. Виджет со списком названий файлов (префиксов)

Окно со списком фотографий с определенным префиксом:

1. Виджет удаления фотографий

2. Виджет добавления комментариев

3. Виджет обработки фотографий

Окно добавления комментариев:

1. Виджет для ввода комментария к фотографии

2. Виджет сохранения комментария

Окно обработки фотографий:

1. Виджет выбора типа обработки (геометрические преобразования, фильтры)

2. Виджет выбора конкретной обработки из выбранного типа

3. Виджет подтверждения обработки фотографии

Окно просмотра обработанной фотографии:

1. Виджет сохранения фотографии

#### Описание БД

В базе данных хранится информация о наименованиях файлов фотографий, местах фотографирования, дате и время фотографирования, комментарии к фотографии. Каждый из этих данных представляет собой отдельную таблицу. То есть таблица наименований файлов, мест, дат и времен, комментариев.
