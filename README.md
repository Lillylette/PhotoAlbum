Пояснительная записка к проекту «Фотоальбом»

Тз.md - файл, в котором расположено техническое задание проекта
test_data – папка, где расположены данные для тестировки программы
src – папка, где находится исходный код. Каждый файл является отдельным классом, где название файла = название класса
main.py – основной файл с кодом
README.md – файл описания проекта

Инструкция по запуску программы 

Для запуска программы необходимо :
1. Клонировать репозиторий к себе на компьютер.
2. Открыть редактор кода
3. В редакторе кода открыть папку проекта
4. Активировать виртуальное окружение
5. Запустить файл main.py
